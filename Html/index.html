Create Token

Before using other APIs, you need to create a token using your Client ID and Secret ID. If you do not have a Client ID and Secret ID, you must first register as an agency. This token will expire after 30 minutes. When the token expires, you will need to create a new one. The generated token will be used as the Bearer Token.

Body Parameter

  • client_id

    - required

    - text

  • secret_key

    - required

    - text

POST

/create/token

https://sitesettingsapi.com/api/v1/developer/create/token
